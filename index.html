// Features

Enterprise-Grade Protection

Everything you need to secure your AI agents at scale

🧠

Multi-Layer Detection

Pattern matching → ML embeddings → adversarial detection → behavioral heuristics. Four layers of defense with 500+ curated attack signatures.

🔐

PASETO Tokens

Cryptographic single-use tokens with replay protection. v4 local encryption with configurable TTL and JTI validation.

⚡

Distributed Rate Limiting

Redis-backed sliding window with per-user, per-IP, and global limits. Automatic adaptive penalties for repeat offenders.

🔍

PII & Secret Scanning

Detect and block credit cards, SSNs, API keys, emails, and more. Auto-redaction available with configurable patterns.

📊

Real-Time Dashboard

Live security monitoring with quarantine review UI. WebSocket events, threat analysis, and attack pattern visualization.

🔗

Framework Plugins

Native integrations for LangChain, LangGraph, CrewAI, and AutoGen. Drop-in guards for any agent framework.

✅

Compliance Ready

SOC2 Type II report generation, GDPR data export/erasure APIs, audit log exporters, and DPA templates included.

🎯

Red Team Engine

Automated adversarial prompt generation for continuous testing. Daily attack corpus updates with mutation-based fuzzing.

🖼️

Multimodal Scanning

Scan images (CLIP + OCR + QR), audio (Whisper), and PDFs. Detect hidden prompts in any media format.
